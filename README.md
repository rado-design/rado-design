## 🧑‍💻 About me

I'm a **Fullstack Engineer** based in **Antananarivo, Madagascar 🇲🇬**, specialized in building robust backend systems and clean REST APIs with **Django REST Framework**, paired with modern frontends in **React / Next.js**.

Currently building [**FiscaCompta**](https://github.com/rado-design) — a full accounting platform following the Malagasy PCG (Plan Comptable Général), covering invoicing, journal entries, TVA declarations, fixed asset management, fiscal year closing, and financial reporting.
